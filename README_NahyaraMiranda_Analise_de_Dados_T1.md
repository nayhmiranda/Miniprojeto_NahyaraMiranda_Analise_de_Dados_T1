# README_NahyaraMiranda_Analise_de_Dados_T1

## ▶️ Como executar o projeto

### Pré-requisitos
- Python 3.12.1 instalado

### Passo a passo

1. Clone o repositório:
git clone https://github.com/nayhmiranda/Miniprojeto_NahyaraMiranda_Analise_de_Dados_T1

2. Instale as dependências:
pip install -r requirements.txt

3. Execute o script:
python Miniprojeto_NahyaraMiranda_Analise_de_Dados_T1.py

> ⚠️ O arquivo `Base Varejo.csv` deve estar na mesma pasta do script.
