# 📊 Miniprojeto_NahyaraMiranda_Analise_de_Dados_T1

Análise exploratória de uma base de dados de varejo com 830.000 registros, desenvolvida como mini-projeto avaliativo do curso de Análise de Dados — Turma T1 no SENAC.

## 🛠️ Tecnologias
- Python 3.12.1
- Pandas
- NumPy

## 📌 O que será analisado
- Carregamento e inspeção da base de dados
- Número de registros, colunas e tipos de dados
- Análise de perfil de clientes
- Análise de categorias de produtos e vendas

## ▶️ Como executar
```bash
pip install pandas numpy
python Miniprojeto_NahyaraMiranda_Analise_de_Dados_T1.py
```
